Intro to Programming Nanodegree The Basics of the Web and HTML

1. World Wide Web

The world wide web consists of personal computers, servers and the internet that connects the previous two. A browser makes it possible to navigate the internet, and the HTTP protocol is used to connect to servers.

2. HTML

HTML means Hyper Text Markup Language. It is comprised of the following components: i) text content, ii) markup, iii) reference to other documents, and iv) links to other pages.

3. HTML Markup

HTML markup is done by adding various tags to text contents. A combination of tags and text contents is called an element.

4. Opening and Closing Tags v. Void Tags

Some tags require both an opening tag at the beginning and a closing tag in the end. (ex. 'p' and 'a') Click to see what happens when you forgot to add the closing tag. However, there are void tags which only need opening tags. (ex. 'br' and 'img')

5. Inline Tags v. Block Tags

While inline tags (ex. 'br' and 'span') do not create a separate section in the document, block tags (ex. 'p' and 'div') inserts an imaginary box in the document.
